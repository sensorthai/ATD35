## 1. ภาพรวมระบบ

ระบบตรวจวัดปริมาณแอลกอฮอล์ ประกอบด้วย:

| อุปกรณ์ | หน้าที่ |
|---|---|
| จอสัมผัส ATD3.5-S3 | หน่วยควบคุมหลัก + แสดงผล |
| เครื่องอ่านใบขับขี่ | อ่านเลขบัตรประชาชนจากใบขับขี่ |
| เครื่องพิมพ์ USB | พิมพ์ Slip รายงานผล |
| โมดูล DS1307 RTC | เก็บเวลา (มีแบตเตอรี่สำรอง) |

---

## 2. การเตรียมเครื่องก่อนใช้งาน

### 2.1 ต่ออุปกรณ์

1. ✅ ต่อ **เครื่องอ่านใบขับขี่** เข้ากับบอร์ด (สาย TX → GPIO 44)
2. ✅ ต่อ **เครื่องพิมพ์ USB** เข้าพอร์ต USB OTG
3. ✅ ต่อ **โมดูล DS1307 RTC** เข้า I2C (SDA → GPIO 15, SCL → GPIO 16)
4. ✅ เปิดเครื่องพิมพ์ ตรวจสอบว่ามีกระดาษ

### 2.2 เปิดเครื่อง

- จ่ายไฟ 5V ให้บอร์ด ATD3.5-S3
- รอจอแสดงผลเปิดขึ้น (~2 วินาที)

## 3. ขั้นตอนการใช้งาน

### ขั้นตอนที่ 1: ตั้งค่าวันเวลา

เมื่อเปิดเครื่อง จะแสดงหน้า **"ตั้งค่าวันเวลา"**

```
┌─────────────────────────────┐
│      ตั้งค่าวันเวลา          │
│                             │
│   ▲         ▲               │
│  12 : 00                    │
│   ▼         ▼               │
│                             │
│   ▲    ▲    ▲               │
│  28 / 05 / 2026             │
│   ▼    ▼    ▼               │
│                             │
│      [ Confirm ]            │
└─────────────────────────────┘
```

**วิธีใช้:**
- กดปุ่ม **▲ ▼** เพื่อปรับ ชั่วโมง / นาที / วัน / เดือน / ปี
- ตรวจสอบว่าวันเวลาถูกต้อง
- กดปุ่ม **Confirm** เพื่อยืนยัน

> **หมายเหตุ:** หากมีโมดูล DS1307 RTC ต่ออยู่ เวลาจะถูกตั้งค่าอัตโนมัติจากโมดูล ตรวจสอบความถูกต้องแล้วกด Confirm ได้เลย

---

### ขั้นตอนที่ 2: กรุณารูดใบขับขี่

หลังกด Confirm จะแสดงหน้า **"กรุณารูดใบขับขี่"** เป็นเวลา 3 วินาที

```
┌─────────────────────────────┐
│                             │
│          💳                 │
│   กรุณารูดใบขับขี่            │
│                             │
└─────────────────────────────┘
```

จากนั้นจะเปลี่ยนไปหน้าหลักอัตโนมัติ

---

### ขั้นตอนที่ 3: หน้าหลัก — แสดงเลขบัตรประชาชน

```
┌─────────────────────────────┐
│   ตรวจวัดปริมาณแอลกอฮอล์      │
│  ┌───────────────────────┐  │
│  │ เลขบัตรประชาชน         │  │
│  │   1521253654587       │  │
│  └───────────────────────┘  │
│   12:30:45  28/05/2026      │
│                             │
│    [ ยินยอมเป่า ]  (เขียว)    │
│                             │
│    [ ไม่ยินยอม ]   (แดง)     │
│                             │
│  🔴 เครื่องพิมพ์ไม่พร้อม      │
└─────────────────────────────┘
```

**วิธีใช้:**
- **รูดใบขับขี่** ผ่านเครื่องอ่าน → เลขบัตรประชาชน 13 หลักจะแสดงบนจอ
- ตรวจสอบเลขบัตรว่าถูกต้อง
- สังเกตสถานะเครื่องพิมพ์:
  - 🟢 **เครื่องพิมพ์พร้อมแล้ว** — พิมพ์ได้
  - 🔴 **เครื่องพิมพ์ไม่พร้อม** — ตรวจสอบสาย USB / เปิดเครื่องพิมพ์

---

### ขั้นตอนที่ 4: เลือกผลการตรวจ

#### กรณี A: ผู้ถูกตรวจ **ยินยอมเป่า**
- กดปุ่ม **"ยินยอมเป่า"** (ปุ่มสีเขียว)
- เครื่องพิมพ์จะพิมพ์ **Slip Format A**
- จอแสดง "Printing..." → "Success!"

#### กรณี B: ผู้ถูกตรวจ **ไม่ยินยอม**
- กดปุ่ม **"ไม่ยินยอม"** (ปุ่มสีแดง)
- เครื่องพิมพ์จะพิมพ์ **Slip Format B** (มีข้อความ "ไม่ยินยอมเป่า")
- จอแสดง "Printing..." → "Success!"

---

### ขั้นตอนที่ 5: ตรวจคนถัดไป

- หลังพิมพ์ Slip เสร็จ ระบบจะกลับมาหน้าหลัก
- **รูดใบขับขี่** ของผู้ถูกตรวจคนถัดไป
- ทำซ้ำขั้นตอนที่ 3-4

## 4. รายละเอียด Slip ที่พิมพ์

### Slip Format A (ยินยอมเป่า)

```
รายงานผลการตรวจวัดปริมาณแอลกอฮอล์

เลขบัตรประชาชนผู้ถูกตรวจวัด
1521253654587

เวลา : 16:55:48
วันที่ : 28/05/2026
```

### Slip Format B (ไม่ยินยอม)

```
รายงานผลการตรวจวัดปริมาณแอลกอฮอล์
เลขบัตรประชาชนผู้ถูกตรวจวัด
1521253654587

ผู้ถูกตรวจวัด **** ไม่ยินยอมเป่า ****

เวลา : 16:55:48
วันที่ : 28/05/2026
```

## 5. การแก้ไขปัญหาเบื้องต้น

| ปัญหา | สาเหตุ | วิธีแก้ |
|---|---|---|
| จอแสดง "No Printer!" | เครื่องพิมพ์ไม่ได้ต่อ หรือปิดอยู่ | ตรวจสอบสาย USB / เปิดเครื่องพิมพ์ |
| จอแสดง "Print Failed!" | เครื่องพิมพ์ขัดข้อง | ตรวจสอบกระดาษ / รีสตาร์ทเครื่องพิมพ์ |
| เวลาไม่ถูกต้อง | DS1307 ไม่ได้ต่อ หรือแบตหมด | ตั้งเวลาใหม่ หรือเปลี่ยนแบต CR2032 |
| เลขบัตรไม่แสดง | เครื่องอ่านบัตรไม่ได้ต่อ | ตรวจสอบสาย RXD ที่ GPIO 44 |
| เลขบัตรแสดง "-------------" | ยังไม่ได้รูดใบขับขี่ | รูดใบขับขี่ผ่านเครื่องอ่าน |
| รูดบัตรแล้วไม่ขึ้น | รูดเร็ว/ช้าเกินไป หรือบัตรเสีย | รูดใหม่ด้วยความเร็วสม่ำเสมอ |
| จอไม่ติด | ไม่ได้จ่ายไฟ | ตรวจสอบแหล่งจ่ายไฟ 5V |

## 6. คู่มือการอบรมเจ้าหน้าที่

### 6.1 หัวข้อการอบรม

| ลำดับ | หัวข้อ | เวลา (นาที) |
|---|---|---|
| 1 | แนะนำอุปกรณ์และส่วนประกอบ | 10 |
| 2 | การต่ออุปกรณ์และเปิดเครื่อง | 10 |
| 3 | การตั้งค่าวันเวลา | 5 |
| 4 | การรูดใบขับขี่และตรวจสอบข้อมูล | 10 |
| 5 | การเลือกผลตรวจและพิมพ์ Slip | 10 |
| 6 | การแก้ไขปัญหาเบื้องต้น | 10 |
| 7 | ฝึกปฏิบัติจริง | 15 |
| | **รวม** | **70 นาที** |

### 6.2 ขั้นตอนการอบรม

#### ช่วงที่ 1: ทฤษฎี (35 นาที)

**1. แนะนำอุปกรณ์**
- อธิบายส่วนประกอบแต่ละชิ้น
- แสดงจุดต่อสายแต่ละจุด
- อธิบายหน้าที่ของแต่ละอุปกรณ์

**2. การต่ออุปกรณ์**
- สาธิตการต่อเครื่องอ่านบัตร (GPIO 44)
- สาธิตการต่อเครื่องพิมพ์ (USB OTG)
- สาธิตการต่อ RTC (I2C GPIO 15/16)
- เปิดเครื่องและตรวจสอบ

**3. การใช้งาน**
- สาธิตขั้นตอนทั้งหมดตั้งแต่เปิดเครื่อง → พิมพ์ Slip
- อธิบายความแตกต่างของ Slip Format A และ B
- แสดงวิธีอ่านและตรวจสอบ Slip

#### ช่วงที่ 2: ปฏิบัติ (35 นาที)

**4. ฝึกปฏิบัติ**
- ให้ผู้เข้าอบรมลองต่ออุปกรณ์เอง
- ลองเปิดเครื่อง + ตั้งเวลา
- ลองรูดใบขับขี่
- ลองกดยินยอม / ไม่ยินยอม
- ตรวจสอบ Slip ที่พิมพ์ออกมา

**5. จำลองปัญหา**
- ถอดสาย USB เครื่องพิมพ์ → ดูข้อความ "No Printer!"
- ปิดเครื่องพิมพ์ → ลองกดพิมพ์ → ดูวิธีแก้ไข
- รูดบัตรผิดวิธี → ดูว่าระบบแจ้งเตือนอย่างไร

### 6.3 เกณฑ์ผ่านการอบรม

ผู้เข้าอบรมต้องสามารถ:

- [x] ต่ออุปกรณ์ทั้งหมดได้ถูกต้องด้วยตนเอง
- [x] เปิดเครื่องและตั้งค่าวันเวลาได้
- [x] รูดใบขับขี่และตรวจสอบเลขบัตรประชาชนได้
- [x] เลือกผลตรวจ (ยินยอม/ไม่ยินยอม) และพิมพ์ Slip ได้
- [x] แก้ไขปัญหาเบื้องต้นได้อย่างน้อย 3 กรณี
- [x] อธิบายความแตกต่างของ Slip Format A และ B ได้
